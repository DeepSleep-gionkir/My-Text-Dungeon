# 🎮 08. Play Interaction & UX

## 1. 핵심 철학 (Core Philosophy)

> **"Simple Input, Rich Feedback"**

플레이어는 복잡한 컨트롤을 할 필요 없이 **"읽고, 판단하고, 클릭(터치)하는"** 단순한 행동만 수행합니다.
하지만 그 결과는 **역동적인 텍스트 연출**과 **시각적 피드백**으로 풍성하게 보상받아야 합니다.

## 2. 화면 레이아웃 (Layout Composition)

### 2.1 상단: 비주얼 영역 (Visual Zone)

- **카드 (Target)**: 현재 조우한 몬스터/함정/보물의 카드가 중앙에 위치.
  - **Visual Art**: 단순 텍스트가 아닙니다.
    - **Center Icon**: `GiDragonHead`, `GiChest` 등 대형 아이콘(128px)이 중앙에 위치.
    - **Frame**: 등급(Normal/Elite/Boss)에 따라 은색, 금색, 보라색 등 화려한 테두리와 장식 패턴 적용.
    - **Background**: `ENV` 태그에 따라 카드 배경색이나 패턴 은은하게 변화 (용암 지대면 붉은 아지랑이).
  - **VFX**: 피격 시 흔들림(Shake), 등장 시 3D Flip, 사망 시 가루가 되어 사라지는 `Dissolve` 효과.
  - **Gauge**: 카드 하단에 HP/시간 제한 바 표시.

* **진행도 (Progression)**: 상단 끝에 `Room 12 / 25` 형태의 인디케이터.

### 2.2 중단: 서사 및 로그 (narrative Zone)

- **스크롤 가능한 텍스트 창**. 가장 중요한 정보 전달 영역.
- **Rich Text Styling**:
  - **데미지**: <span style="color:red; font-weight:bold">-15 HP</span> (강조됨)
  - **상태이상**: `[기절]` 텍스트에 마우스 오버 시 툴팁 표시.
  - **치명타**: 글자가 커지거나 진동하는 애니메이션 효과.
- **Typing Effect**: 새로운 메시지는 타자기로 치듯 한 글자씩 빠르게 출력되어 몰입감 부여. (스킵 가능)

### 2.3 하단: 커맨드 센터 (Command Center)

- **고정 버튼 아님**: 상황(Context)에 따라 버튼의 내용과 개수가 변하는 **가변형 UI**.
- **버튼 구성 예시**:

| 상황               | 버튼 1 (Main)            | 버튼 2 (Sub)             | 버튼 3 (Utility)       |
| :----------------- | :----------------------- | :----------------------- | :--------------------- |
| **일반 전투**      | ⚔️ **[공격]**            | 🛡️ **[방어]**            | 🎒 **[가방/도주]**     |
| **스킬 사용 가능** | 🔥 **[화염구 (MP 10)]**  | ⚔️ **[평타]**            | 🔙 **[돌아가기]**      |
| **함정 조우**      | 🏃 **[회피 시도 (SPD)]** | 🛠️ **[해제 시도 (LUK)]** | 🛡️ **[몸으로 버티기]** |
| **보물 상자**      | 🗝️ **[열쇠 사용]**       | 👊 **[강제로 부수기]**   | 👁️ **[관찰하기]**      |
| **대화 (NPC)**     | 💬 **[설득하기]**        | 💰 **[거래하기]**        | ⚔️ **[공격하기]**      |

## 3. 인터랙션 디테일 (Interaction Details)

### 3.1 다중 선택지 (Branching Choices)

단순한 버튼뿐만 아니라, 대화나 이벤트에서는 **다지선다형 리스트 UI**가 제공됩니다.

- **Dialogue**:
  - 1. "누구냐! 정체를 밝혀라." (위압감)
  - 2. [조용히 무기를 꺼낸다] (전투 준비)
  - 3. [금화 50G를 건넨다] "지나가게 해주시오." (설득 시도 - LUK 체크)
- 일부 선택지는 특정 조건(직업, 아이템, 스탯)이 충족되어야만 해금(`Unlock`)됩니다.

### 3.2 타겟팅 (Targeting)

- 적이 여러 명(`Squad`)일 때, 버튼을 누르기 전 **카드를 직접 클릭**하여 타겟을 지정.
- 타겟팅된 카드는 테두리가 `Gold` 색상으로 빛남 (`Glow Effect`).

### 3.3 롱 프레스 / 우클릭 (Info)

- **카드 롱 프레스**: 해당 몬스터의 상세 정보(알려진 약점, 현재 걸린 버프/디버프 설명) 팝업.
- **내 캐릭터 롱 프레스**: 내 현재 스탯과 장비 상태 요약 보기.

### 3.4 퀵 슬롯 (Quick Slot)

- 자주 쓰는 소비 아이템(포션 등)은 커맨드 센터 옆 작은 원형 슬롯에 배치하여, 턴 소모 없이 즉시 사용 가능한지(Quick Use) 여부 표시.

## 4. 연출 강화 (Juiciness & Audio)

### 4.1 사운드 시스템 (Audio Feedback)

- **SFX (효과음)**:
  - **UI**: 버튼 호버 시 '틱', 클릭 시 '딱' 하는 기계적/마법적 소리.
  - **Combat**: 타격음(Slash, Blunt), 피격음(Ugh), 치명타(칭!), 회피(슉).
  - **Environment**: 문 열리는 소리, 함정 딸깍 소리, 동전 짤랑거리는 소리.
- **BGM (배경음)**:
  - 던전 진입 시: 긴장감 넘치는 앰비언트(Ambient).
  - 전투 돌입 시: 박진감 넘치는 드럼 비트 전환.
  - 보스전: 웅장한 오케스트라 트랙.

### 4.2 시각적 피드백 (Visual Juice)

텍스트 게임의 지루함을 없애기 위한 "손맛" 요소들입니다.

- **Screen Shake**: 강력한 치명타 피격 시 화면 전체가 덜컹거림.
- **Hit Stop**: 타격 순간 0.05초 정도 멈칫하여 타격감 강조.
- **Dynamic Background**:
  - `ENV_MAGMA` 지역이면 배경에 일렁이는 붉은 안개 효과.
  - `ENV_DARKNESS` 상태면 UI 주변부가 어둡게 처리됨(Vignette).

---

**결론적으로, "텍스트 머드 게임"보다는 "인터랙티브 웹툰"이나 "세련된 로그라이크 UI"에 가까운 경험을 제공합니다.**
