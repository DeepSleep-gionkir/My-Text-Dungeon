# 🎴 Card Category Documentation

던전을 구성하는 모든 카드들의 상세 스펙을 정의하는 폴더입니다.
각 파일은 해당 카테고리의 생성 규칙, 행동 로직, 상호작용 방식, JSON 구조 예시를 포함합니다.

## 📑 분류 (Categories)

### 1. [01_Monster_System.md](./01_Monster_System.md)

> **"전투의 적"**
> 단일(Single), 군단(Squad), 엘리트(Elite) 몬스터의 스탯 생성 규칙 및 결정론적 행동 패턴(AI Logic).

### 2. (Planned) Boss System

> 페이즈 전환, 필살기, 전용 연출.

### 3. [02_Trap_System.md](./02_Trap_System.md)

> **"치명적인 위협"**
> 즉발형(Instant)과 지속형(Environmental) 함정. 감지(Detection), 해제(Disarm), 회피(Dodge) 메커니즘 및 실패 시 리스크.

### 4. [03_Shrine_System.md](./03_Shrine_System.md)

> **"거래와 선택"**
> 5개의 선택지(Option)를 가진 제단. AI 자동 생성 모드와 15개 풀에서 5개를 고르는 직접 조립(Assembly) 모드 지원.

### 5. [04_Treasure_System.md](./04_Treasure_System.md)

> **"위험한 보상"**
> 잠긴 상자, 미믹(Mimic) 판별법, 물리적 파괴 vs 기술적 해제, 그리고 등급별 보상 테이블.

### 6. (Planned) NPC & Quest

> 상인 거래, 퀘스트 부여, 대화 트리.

### 7. (Planned) Rest & Campfire

> 휴식처. 체력 회복 vs 장비 수리 vs 스킬 연마 선택지.
